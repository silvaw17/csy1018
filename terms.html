description
DOCTYPE Defines the document type background A shorthand property for setting all the background properties in one declaration
div Defines a section in a document color Sets the opacity level for an element
a Defines a hyperlink border Sets all the border properties in one declaration
h1 Defines HTML headings border-radius A shorthand property for setting all the four border-*-radius properties
h2 Defines HTML headings bottom Specifies the bottom position of a positioned element
h3 Defines HTML headings clear Specifies which sides of an element where other floating elements are not allowed
h4 Defines HTML headings display Specifies how a certain HTML element should be displayed
section Efines a section in a document float Specifies whether or not a box should float
meta Defines metadata about an HTML height Defines the height of the element
link Defines the relationship between a document and an external resource (most used to link to style sheets) width Sets the width of an element
body Defines the document's body z-index Sets the stack order of a positioned element
br Defines a single line break flex Specifies the length of the item, relative to the rest
head Defines information about the document letter-spacing Increases or decreases the space between characters in a text
header Defines a header for a document or section line-break Specifies how/if to break lines
html Defines the root of an HTML column-fill Specifies how to fill columns
label efines a label for an input element cursor Specifies the type of cursor to be displayed
li Defines a list item @keyframes Specifies the animation code
ul Defines an unordered list list-style-type Specifies the type of list-item marker
input Defines an input control list-style Specifies an image as the list-item marker
title Defines a title for the document animation A shorthand property for all the animation properties (except animation-play-state and animation-fill-mode)
video Defines a video or movie direction Specifies the text direction/writing direction
audio Defines sound content font-weight Specifies the weight of a font
b Defines sound content font-size Specifies the font size of text
p Defines a paragraph font-family Specifies the font family for text
div Defines a section in a document text-shadow Adds shadow to text
